Digital Library Management System

A simple Java-based library management system that allows users to add, remove, and search for books using object serialization.

Features

✅ Add new books to the library

Easily add new books with title, author, and ID information.

✅ Remove books by ID

Quickly remove books from the library using their unique identifier.

✅ Search for books by title or author

Find the books you need with a powerful search functionality.

✅ Save & retrieve book data using BOOKS.dat

All your library data is preserved between sessions using Java Object Serialization.

Download and Usage

To use this application:

  1. Clone this repository: git clone https://github.com/ANSHIKA21M/Digital-Library-Management.git
  2. Navigate to the project directory
  3. Run the JAR file: java -jar DigitalLibrary.jar

Technology Stack

  • Java (100%)
  • Object Serialization for data persistence
